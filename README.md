## 🚀 Быстрый старт




## 🎨 Дизайн-система

### Nordic Minimal Theme

- **Философия:** Минимализм, чистая типографика, лёгкая векторная графика
- **Шрифты:** Cormorant Garamond (заголовки), Inter (текст), JetBrains Mono (код)
- **Цвета:** тёплый белый #FAFAF8, мягкий чёрный #1A1A1A, глубокий синий #2B4C7E
- **Стили:** `src/styles/global.css` — единый файл стилей

**Палитра:**
- **Фон:** `#FAFAF8` — тёплый белый, как бумага
- **Текст:** `#1A1A1A` — мягкий чёрный, комфортный для чтения
- **Акцент:** `#2B4C7E` — глубокий синий, фирменный цвет
- **Поверхности:** `#FFFFFF` — карточки и модальные элементы
- **Рамки:** `#E5E5E0` — тёплый серый, ненавязчивые границы

**Концепция:** Скандинавский минимализм с акцентом на типографику. Чистые линии, много воздуха, векторные акценты. Вдохновлено лучшими дизайнами Awwwards.

### 🏷️ Цвета категорий

| Категория | Цвет | HEX | Класс |
|-----------|------|-----|-------|
| `crypto` | Глубокий синий | `#2B4C7E` | `.cat-crypto` |
| `technical-analysis` | Лесной зелёный | `#3A6B4A` | `.cat-technical` |
| `algo-trading` | Янтарный | `#B8862B` | `.cat-algo` |

**Использование:**
```css
.category-badge {
  background-color: var(--cat-crypto);
  color: var(--color-text);
}
```

## 🌍 Интернационализация

Блог поддерживает 3 языка:
- 🇷🇺 Русский (RU)
- 🇬🇧 English (EN)
- 🇪🇸 Español (ES)

Контент хранится в `src/content/blog-[lang]/`

**Структура статьи:** одна папка = одна статья; внутри только `index.md` (и при необходимости `assets/` для картинок). Frontmatter: `title`, `description`, `pubDate`, `category`.

**📝 Руководство по написанию статей:** [`docs/CONTENT_WRITING_GUIDE.md`](docs/CONTENT_WRITING_GUIDE.md) — структура, размер, читаемость, SEO, чек-лист.

**Рабочая ветка:** `main`.

---

## 📝 Контент

Статьи пишутся в Markdown формате в папке `src/content/blog-[lang]/[slug]/`

**Структура статьи:**
```markdown
---
title: "Заголовок"  # До 100 символов, H1
description: "Описание (150-160 символов для SEO)"
pubDate: '2026-01-15'  # Формат: YYYY-MM-DD
category: 'crypto'  # Одна из 4 категорий
---

# Заголовок статьи

Контент статьи...
```

## 🔒 Безопасность

**⚠️ ВАЖНО:** В публичный репозиторий коммитятся только файлы, необходимые для функционирования блога.

- Планы, стратегии и внутренняя документация хранятся в [приватном репозитории](https://github.com/Hampfree-hub/hampfreeblog-private)
- API ключи и секреты не должны попадать в репозиторий
- **Коммиты:** сообщения — на английском. Чтобы не светить личную почту в истории: `git config user.email "Hampfree-hub@users.noreply.github.com"`

## 📍 Деплой

Блог автоматически деплоится на GitHub Pages через GitHub Actions при пуше в ветку `main`.

### OG изображения

OG изображения автоматически генерируются при сборке и деплоятся вместе с сайтом:
- **Генерация:** `scripts/generate-og-images.mjs` (запускается после `npm run build`)
- **Хранение:** `dist/og/` (не коммитится, генерируется при сборке)
- **Доступ:** Автоматически подтягиваются в Telegram/соцсетях через meta tags

## 📊 Аналитика

### GA4 Отслеживание

- **Событие:** `referral_banner_click` (клики по реферальным баннерам)
- **Параметры:** `banner_id`, `banner_name`, `article_category`, `article_lang`, `article_path`
- **UTM-метки:** Автоматически добавляются ко всем реферальным ссылкам

### Реферальные баннеры

- **Автовставка:** После 2-го H2 в каждой статье
- **Баннеры:** Veles, Bybit, BingX, Bitget, FinBazar, Default
