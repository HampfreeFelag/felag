# 📝 Изменения от 25 марта 2026 г.

## Что было изменено

### 1. Добавлена команда `build:full` в `package.json`

**Файл:** `package.json`

```json
{
  "scripts": {
    "build:full": "npx astro build && npm run generate-og"
  }
}
```

**Зачем:**
- Автоматическая генерация OG изображений после сборки
- Безопасность: если Astro сборка упала, OG не генерируется
- Гибкость: можно использовать `npm run build` для быстрой сборки без OG

**Бэкап:** `package.json.backup`
